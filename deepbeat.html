DeepBeat AI

DeepBeat AI is a final Bootcamp project created in just 2 weeks.
It's main goal was to create a product that will allow recognising wherever the ECG result is normal or abnormal. We wanted to make a tool that will increase the accuracy and precision of the diagnosis using Machine Learning.

We've used input data available online as .mat and .dat files and transformed them into images in .jpg format. Those images have been stored on Google Storage in order to be easly accesible by the application. In order to augment the amount of data we've executed some preprocessing functions that added noise and turned the rotation of images. We've used a pretrained ResNet18 model using PyTorch. ResNet is a convolutional neural network that is 18 layers deep. The model has been trained on a Virtual Machine created on Google Cloud and saved it's results on Google Storage in order to be easily accesible from the API created for the prediction process. In order to make predictions we've created a simple Website in Streamlit, that allowed uploading an image, made request to the API deployed on Docker and returned the information about the diagnosis together with precision interval. For better visualization of the results we used Grad Cam in order to show the areas in the image that influenced final score the most.
