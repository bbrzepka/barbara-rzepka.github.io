Sea level web scraping

Sea level web scraping is a task, which goal was to obtain publicly available oceanographic data, available on the spanish website of Puertos de Estado. Data may be downloaded in a form of csv from Ports website, but in order to automate the process and download the information daily, I've created a python script.
I've developed scraping code using Selenium and Pandas library in Python. I've used Big Query on Google Cloud in order to store the information in a dataset.
